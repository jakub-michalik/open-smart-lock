# Example: nanoBoard servo retrofit

> **Status:** placeholder. A concrete end-to-end reference build: openMatterSmartLock running on a nanoBoard (nRF52840) module driving an SG90-class servo connected to a thumb-turn lock via 3D-printed gears.

## Planned content

- Bill of materials (nanoBoard, servo, optional LiPo, wiring, fasteners)
- Mechanical adapter STL files (servo-side and lock-side gears, mounting bracket)
- Wiring diagram
- Build command and `west` workspace setup specific to this example
- Calibration walkthrough
- Demo video link

This example is intentionally a complete, reproducible target build. Anyone with the listed parts and a 3D printer should be able to follow it to a working Matter door lock retrofit.
